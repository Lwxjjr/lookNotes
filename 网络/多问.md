# 网络OSI模型、TCP/IP模型
1. 为了解决多种设备能通过网络相互通信
2. 为了解决各种不同设备在网络互联中的兼容性

![[1721097268006-12c71992-11b3-4a8e-9d05-35ad3e0ab86e.webp]]

OIS七层模型：
- 应用层：负责给应用程序提供统一的接口（HTTP、SMTP、FTP、DNS）
- 表示层：处理数据编码、加密和压缩，为应用层准备数据
- 会话层：打开和关闭两个设备之间的通信
- 传输层：处理两个设备之间的端到端的通信（TCP/UDP）
- 网络层，负责数据的路由、转发、分片
- 数据链路层，负责数据的封帧和差错检测，以及 MAC 寻址；
- 物理层，负责在物理网络中传输数据帧

TCP/IP四层模型：
- 应用层：支持 HTTP、SMTP 等最终用户进程
- 传输层：处理主机到主机的通信（TCP、UDP）
- 网络层：寻址和路由数据包（IP 协议）
- 链路层：通过网络的物理电线、电缆或无线信道移动比特
